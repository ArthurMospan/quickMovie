# Firebase: щоб працювала синхронізація і спільний вішліст

Кнопка «Зберегти» тепер працює ЗАВЖДИ (зберігає на телефоні миттєво).
Але щоб збереження синхронізувались між пристроями і працював СПІЛЬНИЙ вішліст
з партнером — Firestore має приймати записи. Для цього разово:

## 1. Увімкнути анонімний вхід

console.firebase.google.com → твій проєкт → **Authentication** → вкладка
**Sign-in method** → **Anonymous** → **Enable** → Save.

## 2. Правила Firestore

**Firestore Database** → вкладка **Rules** → заміни все на це → **Publish**:

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      allow read, write: if request.auth != null;
    }
  }
}
```

## 3. Перевірка

Відкрий апку через бота → тисни ❤ на будь-якому трейлері → у профілі лічильник
«У списку» має збільшитись, а внизу профілю статус має бути «Синхронізовано».
Якщо пише «Локальний режим» — значить пункт 1 або 2 не зроблено (відкрий
консоль браузера, там буде причина).

## Як працює спільний вішліст

1. Ти в профілі тиснеш «Надіслати запрошення» → друг відкриває посилання.
2. Зв'язок створюється ОДРАЗУ В ОБИДВІ сторони — у тебе і в друга у профілі
   з'явиться картка з ім'ям та аватаркою партнера.
3. Вкладка Watchlist → «Спільні» показує фільми, які зберегли ВИ ОБОЄ.
